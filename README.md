# CVision

# Admin Interface - Screenshots

This branch contains screenshots of the admin dashboard and all management pages.

## [Dashboard](admin_dashboard.jpeg)
Main dashboard showing key metrics: total candidates, companies, job offers, applications, and pending company approvals.

## [Analytics](admin_analytics.jpeg)
Analytics page with charts and statistics about the platform, including top 5 job domains by number of offers.

## Companies Management
- [**Companies List:**](admin_companies.jpeg) Table showing all registered companies with their status (pending, approved, rejected), search bar, and filters.
- [**Company Profile:**](company_profile.jpeg) Detailed view of a company's information and uploaded documents (RC, NIF, additional files), with Approve and Reject buttons.
- [**Company Approve:**](company_approve.jpeg) Popup or confirmation page shown when clicking the approve button.
- [**Company Reject:**](company_reject.jpeg) Popup or confirmation page shown when clicking the reject button.

## [Candidates Management](admin_candidates.jpeg)
List of all candidates with search bar and filters.

## Users Management
- [**Users List:**](admin_users.jpeg) Table showing all users with their roles (candidate, company, admin), search bar, filters, and suspend/reactivate buttons.
- [**User Suspend:**](user_suspended.jpeg) Confirmation popup shown when suspending a user.

## [Job Offers](admin_job.jpeg)
List of all job offers posted on the platform with search bar and filters.

## [Applications](admin_applications.jpeg)
Table showing all applications with: candidate name, job title, company name, compatibility score, test status, test score, and application status (pending, accepted, rejected). Includes search bar and filters.

## Ads Management
- [**Ads List:**](admin_ads.jpeg) Table showing company ad requests with their status (pending, approved, rejected), search bar, and filters. Includes a button to view the ad.
- [**Ad Review:**](admin_ad_review.jpeg) Popup or page showing the ad image and link, with Approve and Reject buttons.

## Separation Requests
- [**Separation Requests List:**](admin_separation.jpeg) Table showing all separation requests from candidates with their status (pending, approved, rejected), search bar, filters, and action buttons (view, approve, reject).
- [**Separation Request Details:**](separation_request.jpeg) Page showing candidate information, their message, and the uploaded proof document.

## Settings
- [**General Settings:**](admin_setting_general.jpeg) Configure footer text and support email.
- [**Platform Settings:**](admin_setting_platform.jpeg) Configure max job offers per company and max applicants per job offer.
- [**Notification Settings:**](admin_setting_notification.jpeg) Enable or disable email notifications to the admin for new company registrations and new candidate registrations.
- [**Security Settings:**](admin_setting_security.jpeg) Change admin password.
