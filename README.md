# CVision

# Candidate Interface - Screenshots

This branch contains screenshots of the candidate dashboard and all related pages.

## [Dashboard](candidate_dashboard.jpeg)
Shows statistics and charts, saved jobs section, recent applications table (with compatibility score, status, and view job button). When accepted for one or more jobs, a "Confirm Offer" button appears allowing the candidate to choose one job. Also displays recommended jobs.

> **Note:** Once a candidate confirms an offer, all other applications are automatically rejected to prevent conflicts. The candidate then gains access to the company's training program and can request separation from the admin.

## [Separation](application_dashboard.jpeg)
- [**Request Separation Button:**](separation.jpeg) Opens a popup window with a file upload field, additional comments text area, and a submit button to send the request to the admin.

## [Profile](candiadte_profile.jpeg)
Allows the user to enter and edit personal information, skills, education, experience, languages, and upload a profile photo and certificate.

## [Job Search](candidate_job.jpeg)
Displays all job offers with search bar and filters. Each job card shows the compatibility score. Sorting options include: Best Match, Most Recent, and Most Popular (based on application count).

## [Job Details](candidate_job_review.jpeg)
Shows complete job information with options to save the job or click "Apply Now".

## Job Apply
- [**With Test:**](job_apply%20+test.jpeg) If the job has an associated test, the application page displays the test information.
- [**Without Test:**](job_apply.jpeg) If no test is required, the application is submitted directly.

## [Tests](candidate_test.jpeg)
Lists all tests taken by the candidate, including pending tests not yet attempted.

## [Test Start](test_start.jpeg)
Displays information about a pending test with a "Start Test" button to begin.

## [Test Attempt](test_attempt.jpeg)
The test appears as multiple-choice questions (MCQ) with a countdown timer.

## [Test Submit](test_sent.jpeg)
The submission page after completing the test. (Note: Tests are taken during the application process, not after hiring, and are used to evaluate the candidate.)

## [Test Results](test_result.jpeg)
Displays the test score and whether the candidate passed or failed.

## [Training](candidate_training.jpeg)
After being hired, the candidate sees all training programs assigned by the company.

## [Training Start](training_start.jpeg)
Training is presented as modules. Each module may contain images, files, videos, text, or MCQ quizzes.

## [Messages](messages.jpeg)
Shows conversation threads between the candidate and companies. The candidate can send messages, emojis, and files. (Note: The company initiates the conversation; the candidate cannot start a conversation.)

## Settings
- [**Notification Settings:**](candidate_setting_notification.jpeg) Enable or disable email notifications for application accepted and application rejected.
- [**Security Settings:**](candidate_security.jpeg) Change password.
- [**Account Settings:**](candidate_setting_account.jpeg) Delete account button.
- [**Delete Account:**](candidate_delete.jpeg) A popup window asks the candidate to type "DELETE" as confirmation before deleting the account.
